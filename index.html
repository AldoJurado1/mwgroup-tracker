MW.GROUP — Configuración inicial

Para sincronizar con Google Sheets necesitas publicar el Apps Script como Web App y pegar la URL aquí.

1
Abre tu Google Sheet → Extensiones → Apps Script
2
Borra el contenido, pega el código del archivo Apps Script adjunto y guarda (Ctrl+S)
3
Clic en Implementar → Nueva implementación → tipo Aplicación web
Ejecutar como: Yo · Acceso: Cualquier usuario
4
Copia la URL de implementación y pégala abajo
La URL termina en /exec — no la de edición
Usar sin sincronización →
